CSS

CSS classes to help write consistent and maintanable code, based on Google Style Guide patterns and SMACSS.

Flexbox structure

```html .flex [ set a div to display flex; use .flex-important if needed ] .flex-grow [ add flex = 1 to a div ] .flex-grow-x [ add flex property to a div, where x is a number from 1 to 10 ] .flex-x [ add flex property to a div, where x is a number from 1 to 10 ] .flex-basis-x-pc [ add flex basis property to a div, where x is 50 or 100 (per cent) ] .column [ add this class to set direction as column ] .column-tablet [ add this class to set direction as column in tablet resolution ] .column-mobile [ add this class to set direction as column in mobile resolution ] .column-mobile-small [ add this class to set direction as column in mobile small resolution ] .row [ add this class to set direction as row ] .row-tablet [ add this class to set direction as row in tablet resolution ] .row-mobile [ add this class to set direction as row in mobile resolution ] .row-mobile-small [ add this class to set direction as row in mobile small resolution ] .wrap [ add this class to set flex wrap ] .wrap-tablet [ add this class to set flex wrap in tablet resolution ] .wrap-mobile [ add this class to set flex wrap in mobile resolution ] .wrap-mobile-small [ add this class to set flex wrap in mobile small resolution ] ```

Alignment

```html .align-items-x [ add this class to align items, where x = center|stretch|start|end ] .justify-content-x [ add this class to justify content, where x = space-around|space-between|center|start|end ] .flex-justify-align-center [ add display flex, justify content center and align items center ] .align-items-middle [ align an element in the middle of the page with view height and view width ] .button-wrapper [ align two buttons in opposit directions and standard margin top, to be used in the end of forms ] .va-middle [ align items to middle using vertical-align ] ```

Standard elements

```html .radius-std [ add border radius standard to an element] .bg-std [ add background standard to an element] .shadow-std [ add box shadow standard to an element] .margin-std [ add standard margins on top, bottom, left and right of an element ] .margin-[top|bottom|left|right]-std [ add standard margin to an element ] .padding-std [ add standard padding on top, bottom, left and right of an element ] .padding-[top|bottom|left|right]-std [ add standard padding to an element ] .[top|bottom|left|right]-std [ set position to an element, where x = "auto" or a number from 0 to 32 multiples of 4 ] ```

Cursor

```html .default [ add cursor default to an element] .pointer [ add cursor pointer to an element] .notallowed [ add cursor notallowed to an element] .move [ add cursor move to an element] ```

Display

```html .hide, .hidden [ add display none to an element ] .show [ add display block to an element ] .inline [ add display inline to an element ] .inline-block [ add display inline-block to an element ] .grid [ add display grid to an element ] .table [ add display table to an element ] .mobile-hide [ add display none to an element on mobile resolution ] .tablet-hide [ add display none to an element on tablet resolution ] .desktop-hide [ add display none to an element on desktop resolution ] ```

Margin

```html .margin-x [ add margin to an element, where x = "auto" or = a number from 0 to 50 multiples of 2 ] .margin-[top|bottom|left|right]-x [ add margin to an element, where x = "auto" or = a number from 0 to 50 multiples of 2 ] .margin-[top|right]-minus-x [ add negative margin to an element, where x = a number from 0 to 50 multiples of 2 ] .margin-y-x [ add margins top and bottom to an element, where x = "auto" or = a number from 0 to 50 multiples of 2 ] .margin-x-x [ add margins left and right to an element, where the second x = a number from 0 to 50 multiples of 2 ] .elements-margin-[right|left] [ add a standard margin right or left to all elements inside a div ] ```

Padding

```html .padding-x [ add padding to an element, where x = a number from 0 to 32 multiples of 4] .padding-[top|bottom|left|right]-x [ add padding to an element, where x = a number from 0 to 32 multiples of 4] .padding-y-x [ add margins top and bottom to an element, where x = a number from 0 to 32 multiples of 4] .padding-x-x [ add margins left and right to an element, where the second x = a number from 0 to 32 multiples of 4 ] ```

Sizes

```html .width-[x|50-pc|100-pc] [ add width to an element, where x = a number from 5 to 750 multiples of 5 ] .max-width-[x|50-pc|90-pc|100-pc] [ add max width to an element, where x = a number from 5 to 750 multiples of 5 ] .min-width-[x|50-pc|100-pc] [ add min width to an element, where x = a number from 5 to 750 multiples of 5 ] .height-[x|100-pc] [ add height to an element, where x = a number from 5 to 750 multiples of 5 ] .max-height-[x|100-pc] [ add max height to an element, where x = a number from 5 to 750 multiples of 5 ] .min-height-x [ add min height to an element, where x = a number from 5 to 750 multiples of 5 ] ```

Positions

```html .relative [ set position relative to an element ] .absolute [ set position absolute to an element ] .fixed [ set position fixed to an element ] .absolute-right [ set position absolute with right = 0 to an element ] .absolute-top-right [ set position absolute with top = 0 and right = 0 to an element ] .absolute-top-left [ set position absolute with top = 0 and left = 0 to an element ] .[top|bottom|left|right]-x [ set position to an element, where x = "auto" or a number from 0 to 32 multiples of 4 ] ```

Colors

```html .[primary|secondary|black|grey|disable|ripple|hover|rag-red|rag-amber|rag-green|white]-color [ set variables for colors to an element ] .[primary|secondary|black|grey|disable|ripple|hover|rag-red|rag-amber|rag-green|white]-bgcolor [ set variables for background color to an element ] .[primary|secondary|black|grey|disable|ripple|hover|rag-red|rag-amber|rag-green|white]-border-color [ set variables for border color to an element ] ```

Formating text classes

```html .bold [ set bold text to an element ] .bold-hover [ set bold text to an element on hover ] .nobold [ remove bold text of an element ] .nobold-hover [ remove bold text of an element on hover ] .italic [ set italic text to an element ] .italic-hover [ set italic text to an element on hover ] .noitalic [ remove italic text of an element ] .noitalic-hover [ remove italic text of an element on hover ] .underline [ set underline text to an element ] .underline-hover [ set underline text to an element on hover ] .nounderline [ remove underline text of an element ] .nounderline-hover [ remove underline text of an element on hover ] .lowercase [ set lowercase text to an element ] .uppercase [ set uppercase text to an element ] .break-word [ set break-word text to an element ] .break-newlines [ command element to obey newlines ('\r\n') ] .no-wrap [ set white space no-wrap text to an element ] .ellipsis [ set ellipsis to an element ] ```

Breakpoints for media queries

```html $media-mobile-smaller [ 400px ] $media-mobile-small [ 500px ] $media-mobile [ 700px ] $media-tablet-min [ 701px ] $media-tablet [ 960px ] $media-desktop-min [ 961px ] $media-desktop [ 1440px ] $media-large-desktop [ 1441px ] ```
